# 资料目录说明

- 论文题目：{{PAPER_TITLE}}

固定子目录：

- `references/`：论文、书籍、报告、政策文件、PDF
- `notes/`：读书笔记、摘录、结构化摘要
- `cases/`：案例材料、新闻整理、事件时间线
- `data/`：表格、统计数据、数据说明
- `images/`：图表、截图、示意图

辅助输入文件：

- `reference_leads.md`：当你只有标准引文、DOI、链接、数据库条目或网页地址时，把线索写在这里；助手会据此主动检索资料

使用规则：

- 在整个工作流执行过程中，应优先查看这里的内容
- 若你补充新材料，直接放进对应子目录即可
- 若你只有文献线索而没有全文文件，可填写 `reference_leads.md`
- 用户不需要指定每份资料用于哪个章节或步骤
- 助手负责理解资料内容，并判断其与论文结构之间的支撑关系
- 对 `reference_leads.md` 中的线索，助手应主动检索、补全元数据，并在 `source_index.md` 中记录结果
